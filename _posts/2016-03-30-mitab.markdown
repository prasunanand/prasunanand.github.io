I built a ruby-gem "mitab" that helps in parsing a PSI-Mitab file.

Installing the gem:
{% highlight bash %}
$ gem install mitab
{% endhighlight %}


Working with the gem:
{% highlight ruby %}
require 'mitab'

text = open(filename) { |f| f.read }
m = Mitab::MitabParser.new(text)

m.print
puts m.mitab
puts m.nodes
puts m.scores
puts m.links
{% endhighlight %}
