# Zasper Blog


## Install jekyll

```bash
gem install bundler jekyll
```


## Clone the repo and install dependencies.

```bash
git clone https://github.com/ZasperIO/zasper_blog
cd zasper_blog/
bundle install
bundle add webrick
```

```bash
bundle exec jekyll serve
```
